# TacticalLab

TacticalLab es un proyecto de análisis táctico y de rendimiento futbolístico
desarrollado en Python.

El objetivo es analizar datos de partidos, eventos y video para generar
insights tácticos y métricas avanzadas.

## Estructura del proyecto

- src/: código fuente
- notebooks/: análisis exploratorio
- data/: datos crudos y procesados
- docs/: documentación
- tests/: tests automáticos

## Entorno

Python 3.11  
Entorno virtual con venv
